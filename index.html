Quick Start

A minimal flow to get immediate value from ScriptSense.

  1. Install ScriptSense in Roblox Studio.
  2. Open any script ‚Äî the live analysis widget appears automatically.
  3. Run scan from the terminal (optional) to scan the whole place.
  4. Use rename to safely rename instances.

Core Concepts

Understanding how ScriptSense views your project.

  • The Project Graph: ScriptSense maps dependencies between ModuleScripts to detect cycles and architectural issues.
  • Live Analysis: As you type, the plugin checks for valid object paths (e.g., workspace.Part.Child) and alerts you if they don't exist.
  • Crash Proofing: Background snapshots ensure you never lose work during a Studio crash.

Command Reference

Access these commands via the ScriptSense Terminal inside Roblox Studio.

scan Analysis

Runs the Path Doctor on the entire project or a specific target. Detects broken references and missing instances.

Usage:
scan [Optional: TargetName]
rename Refactor

Safely renames an Instance. ScriptSense will scan your entire codebase and update any require() or path references to match the new name.

Usage:
rename [OldName] [NewName]
test Utility

Generates a TestEZ-compatible .spec file for the target module. Scaffolds tests for all exported functions.

Usage:
test [ModuleName]
report Feedback

Opens the Feedback GUI to send bug reports or suggestions directly to the developers.

Usage:
report [message]
clear Utility

Clears the terminal output history.

Usage:
clear
