Welcome to ScriptSense

ScriptSense is a project-aware static analysis and workflow tool for Roblox Studio. It watches your place as a system—not just individual scripts.

Core Promise: Prevent broken references and perform safe refactors across your entire project universe.

Installation

ScriptSense works best when installed directly via the Plugin Marketplace.

Plugin Marketplace

The easiest way to get started. Updates are handled automatically by Studio.

Get Plugin

Quick Start

Once installed, open the ScriptSense terminal from the Plugins tab.

  1. Type scan in the terminal and hit enter.
  2. Review the output in the terminal.

Key Features

Feature Description Status
Path Doctor Detects broken object references (e.g., workspace.Part) Stable
Safe Refactor Renames instances and updates all script references automatically Beta
Dep Graph Visualizes module require cycles Stable

Command Reference

Use these commands in the custom terminal window.

scan Analysis

Runs the Path Doctor on the entire project or specific target.

scan [TargetName]
rename Refactor

Safely renames an Instance and updates all Lua source files.

rename [OldName] [NewName]
test Utility

Generates a TestEZ-compatible .spec file for the target module.

test [ModuleName]
